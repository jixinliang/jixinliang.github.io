微积分的英语 “Calculus” 源自拉丁语，意思是 “小石头”，因为它是从分析小的部分来了解大的整体。

微分是把整体分拆为小部分来求它怎样改变。

积分是把小部分连接（积）在一起来求整体有多大。

高等数学, 又叫微积分. 如果把微积分的理论体系, 看作是一座大厦的话, 那么在集合概念基础上建立起来的集合论, 便是这座大厦的基础

集合论的难点是如何理解无穷集合

微积分是变量数学, 反映变量与变量之间的依赖关系的是函数, 所以微积分连接的基本对象是函数, 而数理函数的工具是微分和积分. 微分与积分的基础是极限理论,

函数是集合到集合的映射

集合: 具有某种特定性质并且可以彼此区别的事物的总体

元素: 集合里的每一个事物

用大写字母表示集合, 用小写字母表示元素

自然数: N = {x | x = 1, 2, …}

整数: Z = {x | x = 0, +-1, +-2, …}

有理数: Q = {p/q | p, q ∈ Z, q != 0}

实数: R

复数: C

集合的表示法

  • 列举法: 按任意顺序列出集合的所以元素, 并用花括号括起来, A={1, 2, 3} 有限集合
  • 描述法: 设P(a)为某个与a有关的条件或法则, A为满足P(a)的一切a构成的集合, 则即为A={ a | P(a) }. A={x | x = 2n+1, n ∈ N}

直积: 笛卡尔积

微积分

数学之美: 和谐, 简洁, 对称

实数的性质:

  • 有序性
  • 连续性或完备性

数集的上界和上确界:

设E是一个非空实数集, M是一个实常数, 如果对于E中的任何元素x, 均有 x <= M, 则称M为数集E的一个上界, 并称E有上界

如果一个实数E有上界, 称E的最小上界为上确界, 即为supE(supremum)

连续性公理: 一个非空有上界的实数必有上确界

公理: 不需要的证明的事实

设E是一个非空实数集, m是一个实常数, 如果对于E中的任何元素x, 均有x >= m, 则称m为数集E的一个下界, 并称E有下界

如果一个实数集E有下界, 称E的最大下界为下确界, 即为infE(infimum)

一个非空有下确界的实数集必有下确界

有限区间

  • 开区间: (a, b) = {x | a < x < b}

  • 闭区间: [a, b] = {x | a <= x <= b}

  • 半闭区间: [a, b) = {x | a <= x < b}; (a, b] = {x | a < x <= b}

无限区间

  • 无限开区间: (-∞ , a) = {x | x < a}; (a, +∞ ) = {x | x > a}

  • 无限闭区间: (-∞, a] = {x | x <= a}; [a, +∞) = {x | x >= a}

符号链接

全体实数的集合: R = (-∞, +∞)

邻域: 以点a为中心的任何开区间, 记作U(a)

δ邻域: U(a, δ) = (a - δ, a + δ) = {x | |x - a| < δ}

去心邻域: U(a, δ) = {x | 0 < |x - a| < δ}

左邻域: (a - δ, a)

右邻域: (a, a + δ)

映射是现代数学的基本概念, 它反映了事物之间的一对一和多对一的这样的依赖关系

设A, B是两个非空集合, 若对A中的任一元素x, 依照某种规律f, 恒有B中的唯一确定的元素y与之对应, 则称对应规律f为一个从A到B的映射. 记作f: A -> B

设A, B是两个集合, 若存在一个一一映射φ: A -> B, 则称集A与集B是等势的

两个有限集是等势的, 等且仅当他们的元素个数相等

  • 所有偶数与自然数是一样多的
  • 无理数比有理数的个数多

函数: 起源于描述一个量依赖于另一个量的变化

函数就是集合到集合之间的一个映射

设D是R中的非空子集, 称映射f: D -> R为定义在D上的一元映射

通常记作: y = f(x), x ∈ D

x为自变量, y为因变量

D为定义域

Rf = {f(x) | x ∈ D}为值域

Gf = {(x, f(x)) | x ∈ D}为函数的图形(图像)

莱布尼茨: 函数是变化和运动的“普通语言”

不变和静止的数学 -> 变化和运动的数学

函数它相当于是一个变换的运算器, 使得每一个输入有一个相应的输出

Note:

  • 函数y = f(x), x ∈ D由其定义域D与对应法则f唯一确定, 与自变量, 因变量以及函数的名称无关

  • 两个函数相等是指他们的定义域与对应法则都相等, 自然他们的值域也相同

函数定义域的确定:

  • 自然定义域: 函数表达式在实数域中有意义的所有自变量的集合
  • 实际定义域: 问题的实际背景所要求的自变量的取值范围

考虑函数的时候, 如果这个函数没有给出实际背景, 在讨论其定义域时, 通常讨论的是自然定义域, 如果这个函数有实际背景, 则讨论的是实际定义域

函数的表示法

  • 描述法: 通过语言描述
  • 表格法: 数值表格
  • 图形法: 图形表示
  • 公式法: 精确公式表示

常值函数: y = C

绝对值函数: y = |x| = {x, x >= 0; -x, x < 0}

符号函数: y = sgnx = {1, x > 0; 0, x = 0; -1, x < 0}

取整函数: y = [x], 不超过x的最大整数[x] <= x < [x]+1; [x+1] = [x]+1

狄利克雷函数: D(x) = {1, x ∈ Q; 0, x ∉ Q}

函数的运算

  • 四则运算

    • (f + g)(x) = f(x) + g(x), x ∈ A ∩ B
    • (f * g)(x) = f(x) * g(x), x ∈ A ∩ B
    • (f / g)(x) = f(x) / g(x), x ∈ A ∩ B 且 g(x) != 0

    函数的定义域不能以运算后的形式来确定

  • 复合运算: 函数复合过程就是代入的过程, 复合函数又称为函数的函数

  • 求逆运算

函数有界与无界的几何特点

  • 有界函数的图形介于某两条水平直线之间

  • 无界函数的图形不能介于任何两条水平直线之间

耐皮儿常数e

初等函数: 由基本初等函数与常值函数经过有限次四则运算或复合运算得到的由一个统一的解析式子表示的函数

多项式函数

有理函数: 两个多项式相除

通过对多项式进行代数运算(四则运算与开放运算)所得到函数称为代数函数, 非代数函数的函数称为超越函数

超越函数包括三角函数, 反三角函数, 指数函数, 对数函数等

曲线是一个抽象的数学概念, 它是一个只有长度没有宽度的几何对象, 就是平面或空间中间按一定条件变动的动点的轨迹

对于数列{an}, 若存在常数a, 对于任意给定的正数ε均存在正整数N, 当n > N时, 恒有

|an - a| < ε成立, 则称数列{an}存在极限(收敛), 常数a称为数列的极限, an -> a(n -> ∞)

若上述常数不存在, 则称数列不存在极限(发散)

当一个变量逐次取的值无限趋近一个定值时, 如果最终变量的值与该定值的差要多小就有多小, 那么, 这一定值就称为所有其它值的极限

对于任意给定的两平行线y = a + ε与y = a - ε, 一定可以找到正整数N, 对于n > N的所有点(n, an)均落在这两条平行线之间

设数列{an}存在极限, 则极限唯一
